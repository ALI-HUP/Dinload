# 🦖 Dinload
A Persian platform for downloading movies and series, built with the unique **Dinisor** aesthetic.

---

## 🚀 Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

## 🔗 Deployment

Live Demo: https://dinload.vercel.app

## 🎨 Design System (Dinload Palette)

The **Dinload Design System (DDS)** uses a unified visual theme designed to work beautifully in both dark and light environments without requiring a theme toggle.

It combines Tailwind CSS Slate tones with vivid blue accents to create a modern, elegant, and cinematic interface.

---

## 🎯 Color Hierarchy (Darkest to Lightest)

| Tailwind Class | Hex Code | Usage |
| --- | --- | --- |
| `slate-900` | `#0F172A` | Deep accent / dark text |
| `slate-800` | `#1E293B` | Main background |
| `slate-700` | `#334155` | Surface / card background |
| `sky-600` | `#0284C7` | Primary brand color |
| `sky-500` | `#2B7FFF` | Highlight / lighter accent |
| `slate-400` | `#94A3B8` | Secondary / meta text |
| `slate-100` | `#F1F5F9` | Main heading text |
